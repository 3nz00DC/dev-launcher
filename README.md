# ⌚ Visor de Programação - Terminal Edition

Um projeto front-end criado para simular a interface de um smartwatch com uma estética de terminal de linha de comando. Desenvolvido para praticar manipulação de DOM, animações CSS e design de interfaces minimalistas.

## 🚀 Demonstração

## ✨ Funcionalidades
* **Relógio em Tempo Real:** Captura e formata a hora e a data exatas do sistema.
* **Transição Suave (Slide):** Animação em CSS alternando automaticamente entre a tela principal (tempo) e a tela secundária (saúde) a cada 5 segundos.
* **Simulação de Dados:** Geração dinâmica de batimentos cardíacos simulados via JavaScript para dar vida à interface.
* **Estética Cyber/Hacker:** Tipografia monoespaçada, cores neon (cyan/green/yellow) e cursor intermitente.

## 💡 Casos de Uso
Além de ser uma ótima peça de portfólio para exibir habilidades de Front-End, a interface rodando em tela cheia (F11) serve perfeitamente como elemento de cenário autêntico. Pode ser deixada em um monitor de fundo para compor a atmosfera de uma cena com um hacker trabalhando em produções audiovisuais ou curtas-metragens.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica e organização dos elementos.
* **CSS3:** Estilização, layout Flexbox, bordas arredondadas e animações (`@keyframes`, `transition`, `transform`).
* **JavaScript (Vanilla):** Lógica de atualização de tempo, `setInterval` para as animações e manipulação de classes.

## 💻 Como rodar localmente
1. Faça o clone do repositório.
2. Abra o arquivo `index.html` diretamente no seu navegador.
3. Não é necessária nenhuma instalação adicional de pacotes.
